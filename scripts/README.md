# scripts

이 저장소는 개인적인 Dotfiles 관리를 위한 스크립트 모음입니다.

## 스크립트 목록

1. `bootstrap-omz.sh`: Oh My Zsh를 설치하고 기본 설정을 구성하는 스크립트입니다. 필요한 플러그인과 테마를 설치하고 설정 파일을 준비합니다.
2. `manage-krew-brew.sh`: Krew와 Homebrew 패키지 관리를 위한 통합 스크립트입니다. Krew를 사용하여 Kubernetes 플러그인을 설치하고, Homebrew를 통해 필요한 패키지를 관리합니다. 이 스크립트는 Krew 리스트의 백업과 복원 기능을 포함합니다.
3. `setup-dotfiles.sh`: 개인 Dotfiles 설정을 위한 스크립트로, 사용자 지정 설정 파일을 특정 위치에 심볼릭 링크로 연결합니다. 이 스크립트를 실행하여 사용자 환경을 빠르게 설정할 수 있습니다.
4. `setup-iterm2.sh`: iTerm2의 기본 설정과 사용자 정의를 위한 스크립트입니다. iTerm2에서 사용할 테마와 프로파일을 설정하고, 필요한 플러그인을 설치합니다.
